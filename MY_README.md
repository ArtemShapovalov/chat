# Install

1. install elasticsearch 1.4.4
1. install composer
2. composer update
3. cd to {project}/frontend
    npm install
    bower install
    
4. cd to {project}/node
    npm install
    
5. cd to {project}
    php init
    
start
frontend -> cd to {project}/frontend grunt serve
api      -> cd to {project}/api/web and execute command php -S localhost:8092
